# Business Listing & Rating System

This is a simple Business Listing and Rating System built using **PHP, MySQL, jQuery, AJAX and the Raty Star Rating Plugin**.

The application allows users to manage businesses and submit ratings without page refresh.

---

## Features

1 Add Business  
2 Edit Business  
3 Delete Business  
4 Submit Rating (0 - 5 with half star support)  
5 Average Rating Calculation  
6 Duplicate Email & Phone Validation  
7 Real-time Rating Update  
8 AJAX Based (No Page Refresh)  
9 Star Rating UI using Raty Plugin

---

## Technologies Used

- PHP
- MySQL
- jQuery
- AJAX
- Bootstrap
- Raty Star Rating Plugin
